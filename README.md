# AI Career Advisor Assistant (LLM Mini Project)

An AI-powered career guidance assistant built using **Python, Streamlit, and LLM APIs (OpenRouter)**.  
This mini project demonstrates how Large Language Models can be integrated into a simple web application to provide personalized career advice.

---

## Project Overview

This project is a beginner-friendly exploration of:

- Large Language Models (LLMs)
- Prompt engineering
- API integration with AI models
- Streamlit frontend development
- Secure API key handling using environment variables

The assistant takes user input related to skills or career goals and generates AI-based career suggestions.

---

## Features

✅ AI-powered career advice generation  
✅ Simple Streamlit web interface  
✅ Secure API key handling (.env file)  
✅ Modular backend AI logic  
✅ GitHub-ready project structure  
✅ Beginner-friendly LLM implementation  

---

## Tech Stack

- **Python**
- **Streamlit** – frontend UI
- **OpenRouter API** – access to LLM models (Mistral, Llama etc.)
- **dotenv** – environment variable management
- **Git/GitHub** – version control

---

## How It Works

1. User enters skills or career query.
2. Input is sent to an LLM via OpenRouter API.
3. The AI model processes the prompt.
4. Response is displayed on a Streamlit web interface.
